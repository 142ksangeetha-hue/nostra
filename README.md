# Nostra - E-commerce Website

## 📖 Project Overview

Nostra is a responsive e-commerce website developed using HTML, CSS, and JavaScript. The project demonstrates a modern online shopping interface with multiple pages, product collections, search functionality, and responsive design.

## 🚀 Features

- Responsive Home Page
- Collections Page
- Contact Us Page
- Product Search Functionality
- Category Filter
- Mobile-Friendly Navigation Menu
- Modern UI Design
- Smooth User Experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
